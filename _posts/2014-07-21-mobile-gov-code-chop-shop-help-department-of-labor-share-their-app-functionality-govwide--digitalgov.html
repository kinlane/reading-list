You’ve just found a great open source fed agency app on the Mobile Code Sharing Catalog, and would love to use one of its cool functionalities for your own agency’s app.As federal agencies release more and more code to the open source community, this dilemma is becoming increasingly commonplace.Agencies who open-source their entire app’s code are taking an excellent first step; the next challenge is to get the really interesting and useful code reused more readily.One way to do this is to annotate the app’s code and open source the code via functionality.This is the goal behind our first Mobile Gov Code Chop Shop.
