WordPress has maintained its popularity despite lacking a built-in REST API that enables external applications to easily interact with a WordPress-based websites content and users.That will likely soon change, however.A WordPress plug-in dubbed WP API extends the open source content management system with a full REST API that uses JSON and OAuth authentication.WP API recently hit version 1.0, and its creator, Ryan McCue, is pushing to have it integrated into the WordPress core as early as the next major release, 4.
