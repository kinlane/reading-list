As the way we use technology evolves, so have API architectures.The current rise (and eventual prevalence) of the Internet of Things (IoT) is making it necessary to create new API architectural styles addressing scenarios that go beyond client/server.It is for this reason that a recent study by CA Technologies was particularly interesting.The survey was led by The API Academy, an expert group at CA Technologies that provides consulting on API business strategy, design and architecture best-practices.The survey, which was promoted to the CA customer base, resulted in 180 responses from API management professionals and enterprise architects.
