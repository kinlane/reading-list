Analyst firm Forrester has released the Q3 2014 version of The Wave™ ranking for API Management Solutions.Using their 40 point criteria, Forrester evaluates API management providers, identifying the “11 most significant software providers”— 3scale, Apigee, Axway, CA Technologies, IBM, Informatica, Intel Services, MuleSoft, SOA Software, Tibco Software, and WSO2.The report focuses on how well each vendors products fits within the 40 point Forrester criteria, and where each provider stands in relation to each other.The analysis, combined with three buying guides highlighting key decision factors, aims to help companies select the right API provider for their API management needs.The Wave™: API Management Solutions for Q3 2014 is available on the Forrester website for USD $2495.
