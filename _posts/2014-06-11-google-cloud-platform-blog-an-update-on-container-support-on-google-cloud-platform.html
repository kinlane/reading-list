Everything at Google, from Search to Gmail, is packaged and run in a Linux container.Each week we launch more than 2 billion container instances across our global data centers, and the power of containers has enabled both more reliable services and higher, more-efficient scalability.Now we’re taking another step toward making those capabilities available to developers everywhere.Support for Docker images in Google App Engine Last month we released improved Docker image support in Compute Engine.Today, we’re building on that work and adding a set of extensions that allow App Engine developers to build and deploy Docker images in Managed VMs.
