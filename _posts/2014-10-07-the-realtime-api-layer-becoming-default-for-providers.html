One of the API news story of the day was the release of the Hacker News API.I’m only lightly interested in the fact that HN now has an API because, honestly I dont feel like an API at this point in the evolution of tech curation site is something to applaud, but their approach to making their API real-time I think does subtly reflect other currents in the API space.HN partnered with Firebase to make sure the tech news curation site doesnt just have an API, it has a real-time API.Meaning you can now get HN links in machine readable way, so that you can build the popular flow of tech information into other web, and mobile applications—in real-time!HN launching an API…yawn!
